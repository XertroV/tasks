# The Administrator's Desk

A high-fidelity immersive prototype for "The Backlogs" documentation.

## Concept
You are sitting at a desk in Level 0 of The Backrooms. The hum of the lights is constant. The yellow wallpaper stretches infinitely. This desk is your only safe haven.

## Tech Stack
- **Runtime:** Bun
- **Framework:** React + Vite
- **Styling:** Tailwind CSS v4 + Framer Motion
- **3D:** React Three Fiber (Three.js)

## Setup & Run

1.  **Install Dependencies:**
    ```bash
    bun install
    ```

2.  **Start Development Server:**
    ```bash
    bun run dev
    ```

3.  **Build:**
    ```bash
    bun run build
    ```

## Controls
- **Left Sidebar:** Navigation folders (Intro, Logs, Tasks, Admin).
- **Center:** The current file/paper. Scroll to read.
- **Top Right:** Search index and notifications.
